# Electronics Glossary

IC
- Integrated circuit

DIP
- Dual inline package

Continuity
- Current can flow

Resistor - R
- component used to limit current
- Ohms 

Voltage - V
- Volts

Current - I
- flow of electrons
- Amperes

Ohm's Law
-  V = I * R

LEO GER
: "lost electrons oxidized" vs "gained electrons reduced"

Anode
: electrode where oxidation occurs LEO ... think "Away" from here

Cathode
: electrode where reduction occurs GER ... think "`Comes" here

AN OX
: oxidized at anode

RED CAT
: reduced at cathode 

GPIO
: general purpose input output pins for use

Tx
: transmit

Rx
: receive

Pulse Width Modulation - PWM
: pulse width modulation

System != Embedded Programming
: in embedded often your code is the only code

Real Time Operating System - RTOS
: real time operating system ... concurrency not parallelism, tasks not threads
: provides primitives that Bare Metal does not provide

Bare metal
: only your code, level belown RTOS
: hardware, mmio, pmio, peripherals

Hardware Abstraction Library - HAL
: hardware abstraction library delivers abstraction for registers that contain settings in memory 

Baud Rate
: speed at which information is transmitted over a serial connection in bits per second
